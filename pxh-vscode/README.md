# PXH Language Support

Hỗ trợ highlight và IntelliSense cho ngôn ngữ **PXH** với cú pháp:

```pxh
pxh[css]: {
    p {
        color: hotpink;
    }
}

pxh[html]: {
    <p onclick="message()">Hello, World! - PXH</p>
}

pxh[js]: {
    const message = () => {
        console.log('Hello, World! - PXH');
    }
}
```
